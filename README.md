## ðŸ“± Project Overview

PlantPal is a complete plant care management application designed to help users maintain healthy plants through systematic watering reminders. The app combines intuitive UI design with robust background processing to deliver reliable notifications and seamless plant management. Built using MVVM architecture and leveraging the latest Android Jetpack libraries, PlantPal demonstrates professional mobile development practices suitable for portfolio and university project submissions.

**Key Capabilities:**
- Track unlimited plants with customizable watering schedules
- Receive smart daily notifications at 9 AM for plants needing water
- Capture and store plant photos for easy identification
- View real-time countdown timers showing days until next watering
- Edit watering history with custom "last watered" dates for accurate tracking
